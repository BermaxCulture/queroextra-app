18 · Breakpoints & Layout

19 · Regras de Uso — Proibições
Nunca usar gradientes — o app é flat por design
Nunca usar Inter, Roboto ou Arial — a fonte é DM Sans exclusivamente
Nunca usar font-size abaixo de 16px em inputs (causa zoom iOS)
Nunca usar área de toque menor que 44px
Nunca hardcodar cores — usar sempre os tokens CSS --qe-*
Nunca usar fundo colorido no item ativo do bottom nav — só traço amarelo + cor do ícone
Sempre usar var(--qe-yellow) #F5C000 — nunca variações próximas sem motivo
Sempre usar DM Sans, importada do Google Fonts no projeto
Sempre mobile-first — escrever o CSS base para 375px, adaptar para desktop com media queries
